Willow UI
=========

Client for Willow cloud management and monitoring.

# Get started

Prerequirements

* [Node.js](https://nodejs.org/)

Install dependencies
```
$ npm install
```

Open development environment
```
$ gulp
```

Run tests
```
$ gulp test
```

Create build
```
$ gulp build
```
