Willow UI
=========

Client for Willow cloud management and monitoring.

# Get started

Prerequirements

* [Node.js](https://nodejs.org/)

Install dependencies
```
$ npm install
```

Run tests
```
$ gulp test
```

Create build
```
$ gulp build
```
